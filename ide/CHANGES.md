# Discordia Terminal IDE — Changes

## v2.0 (2026-04-17)

### New Features
- VS Code-style activity bar (Explorer, Search, Git, Build, Languages, API Keys)
- Command palette (Ctrl+Shift+P) with 25+ commands
- Git source control panel — commit & push directly from IDE
- Build log viewer — live GitHub Actions CI/CD status
- Multiple terminal tabs with independent bash sessions
- File search across all workspace files (Ctrl+Shift+F)
- 10 color themes (Matrix, Dracula, Monokai, Nord, Solarized, Cyberpunk, Ocean, Forest, Volcano, Midnight)
- Monaco editor (VS Code engine) with 140+ language support
- Custom language creator (.pepe, .custom, etc.)
- File packaging (ZIP/APK/EXE/Geode/MCAddon/MCPack)
- API key manager
- System info panel
- Keyboard shortcuts: Ctrl+N, Ctrl+S, Ctrl+W, Ctrl+`, F5, Ctrl+Shift+P
- Resizable sidebar and terminal panel
- File diff viewer (compare local vs GitHub)
- Build badge in status bar
- Notification toasts
- Menu bar (File, Edit, View, Git dropdowns)
- Auto-detect project type for packaging
- VS Code breadcrumbs
- Editor tab dirty indicator

### Bug Fixes
- Terminal resize and reconnect
- Null-safe DOM binding
- File tree expand/collapse
- Tab close handling

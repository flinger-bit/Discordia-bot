# Discordia Terminal IDE v2.0

Full VS Code-style web IDE with:
- Activity bar (Explorer, Search, Git, Build, Languages, API Keys)
- Command palette (Ctrl+Shift+P)
- Git source control with commit & push
- GitHub Actions CI/CD build log viewer
- Multiple terminal tabs
- File search (Ctrl+Shift+F)
- 10 color themes
- Monaco editor (140+ languages)
- Custom language creator
- File packaging (ZIP/APK/EXE/Geode/MCAddon/MCPack)
- API key manager
- System info panel
